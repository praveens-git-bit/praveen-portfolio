This repository contains my cloud-native dual-identity portfolio, designed to represent two interconnected worlds:

• Cloud Consultant – focused on Azure architecture, infrastructure design, DevOps automation, AI integration, and production-grade deployment practices.
• Beyond Work – a creative dimension reflecting storytelling, conceptual system thinking, and innovative design exploration.

The project is built using modern frontend architecture (Vite + React + TypeScript) and structured with performance, scalability, and maintainability in mind.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
